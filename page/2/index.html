unable to resolve host м—җлҹ¬ н•ҙкІ° л°©лІ•

Posted on 2018-05-07 In linux

ubuntuм—җм„ң sudoлҘј мӮ¬мҡ©н•  л•Ңл§ҲлӢӨ лӢӨмқҢкіј к°ҷмқҖ л¬ёкө¬к°Җ лңЁлҠ” кІҪмҡ°к°Җ мһҲлӢӨ.

лӘ…л №мқ„ мӢӨн–үн•ҳлҠ” лҚ°м—җ л¬ём ңк°Җ лҗҳлҠ” кІғ к°ҷм§ҖлҠ” м•Ҡм§Җл§Ң л§Өмҡ° кұ°мҠ¬лҰ¬лҜҖлЎң н•ҙкІ°н•ҙліҙлҸ„лЎқ н•ҳмһҗ.

hostname

hostnameмқҖ л„ӨнҠёмӣҢнҒ¬м—җ м—°кІ°лҗң мһҘм№ҳм—җ л¶Җм—¬лҗҳлҠ” кі мң н•ң мқҙлҰ„мқҙлӢӨ.

ubuntu desktop лІ„м „мқ„ м„Өм№ҳн•  л•Ң userлҘј мғқм„ұн•ҳлӢӨліҙл©ҙ кі„м •лӘ…кіј pcмқҙлҰ„мІҳлҹј мқҙмғҒн•ң кІғмқҙ кІ°н•©лҗҳм–ҙ кёёкІҢ hostnameмқҙ л¶ҷкіӨ н•ңлӢӨ.
мқҙ hostnameмқҖ terminalмқҳ лӘ…л №н–үм—җ кі„мҶҚ л…ём¶ңлҗҳлҜҖлЎң кёёл©ҙ кұ°мҠ¬лҰ¬кё° л§Ҳл Ёмқҙлқј, н•„мһҗлҠ” ліҙнҶө devлЎң ліҖкІҪн•ҳлҠ” нҺёмқҙлӢӨ.

/etc/hostnameкіј /etc/hosts

ubuntuмқҳ hostnameмқ„ ліҖкІҪн•ҳл Өл©ҙ мҙқ л‘җ к°ңмқҳ нҢҢмқјмқ„ мҲҳм •н•ҙм•јн•ңлӢӨ.
кІ°лЎ л¶Җн„° л§җн•ҳмһҗл©ҙ unable to resolve host л¬ёкө¬к°Җ м¶ңл Ҙлҗҳм—ҲлҚҳ мқҙмң лҠ” мқҙ л‘җ нҢҢмқјмқҳ лӮҙмҡ©мқҙ лӢ¬лһҗкё° л•Ңл¬ёмқҙм—ҲлӢӨ.

м—җлҹ¬к°Җ л°ңмғқн•ҳлҠ” мғҒнҷ©м—җм„ң л‘җ нҢҢмқјмқ„ л№„көҗн•ҙліҙл©ҙ лӢӨмқҢкіј к°ҷлӢӨ.

/etc/hostnameм—җлҠ” devлқјкі  лӮҳмҷҖмһҲлҠ” л°ҳл©ҙ, /etc/hostsм—җлҠ” ліҖкІҪ м „мқҳ hostnameмқҙ м“°м—¬мһҲм—ҲлӢӨ.
мҳҲм „м—җ hostnameмқ„ л°”кҝҖ л•Ң /etc/hostnameл§Ң ліҖкІҪн•ң лӘЁм–‘мқҙлӢӨ.
кі„м • мғқм„ұ лӢЁкі„м—җм„ң л°”кҝЁмңјл©ҙ мү¬мӣ  мқҚмқҚ

/etc/hostsм—җм„ң мҳҲм „ hostnameмқ„ м§Җмҡ°кі  нҳ„мһ¬ hostnameмңјлЎң ліҖкІҪн•ҙмЈјл©ҙ лҚ” мқҙмғҒ л¬ёкө¬к°Җ м¶ңл Ҙлҗҳм§Җ м•ҠлҠ”лӢӨ.

cronм—җм„ң bash script мӢӨн–үмӢңнӮӨкё°

Posted on 2018-04-28 In linux

rebootн•  л•Ңл§ҲлӢӨ scriptлҘј мӢӨн–үмӢңнӮ¬ мқјмқҙ мғқкІЁм„ң cronмқ„ мӮ¬мҡ©н•ҳкІҢ лҗҳм—ҲлҠ”лҚ°, мӮҪм§Ҳмқ„ н•ҳл©ҙм„ң м•ҢкІҢлҗң лӮҙмҡ©мқ„ кё°мҲ н•ҳкі мһҗ н•ңлӢӨ.
м—¬лҹ¬ к°Җм§Җк°Җ м—®м—¬мһҲлҠ” лӮҙмҡ©мқҙлқј мўҖ мһҘнҷ©н•ҳлҜҖлЎң, л§Ё л§Ҳм§Җл§ү кІ°лЎ л§Ң мқҪм–ҙлҸ„ л¬ҙл°©н•ҳлӢӨ.

cron

cronмқҖ crontabм—җ мҳҲм•Ҫлҗң мһ‘м—…мқ„ л°ҳліөм ҒмңјлЎң мӢӨн–үмӢңнӮӨлҠ” лҚ°лӘ¬мқҙлӢӨ.
cronмқҖ ubuntu кё°ліё нҢЁнӮӨм§ҖмқҙлҜҖлЎң лі„лҸ„мқҳ м„Өм№ҳкіјм •мқҙ н•„мҡ”м—ҶлӢӨ.
лӢӨмқҢмқҳ лӘ…л №м–ҙлҘј мһ…л Ҙн•ҳм—¬ мһ‘м—…мқ„ л“ұлЎқн•  мҲҳ мһҲлӢӨ.

мһ‘м—…мқ„ л“ұлЎқн•  л•ҢлҠ” мӢӨн–үлҗ  мЈјкё°, мӢӨн–үн•ҳкі мһҗн•ҳлҠ” лӘ…л №м–ҙлӮҳ нҢҢмқјмқ„ м Ғмңјл©ҙ лҗңлӢӨ.
rebootн•  л•Ңл§ҲлӢӨ /root/cron/test.shлҘј мӢӨн–үн•ҳкі  мӢ¶лӢӨл©ҙ м•„лһҳмҷҖ к°ҷмқҙ мһ‘м—…мқ„ л“ұлЎқн•ңлӢӨ.

л§ӨлІҲ к»җлӢӨ мјӨ мҲҳ м—ҶмңјлҜҖлЎң н…ҢмҠӨнҠён•  л•ҢлҠ” л§Өл¶„ мӢӨн–үлҗҳлҸ„лЎқ н•ҳмҳҖлӢӨ.

*мқҖ everyмқҳ мқҳлҜёлЎң, л§Ё мҷјмӘҪл¶Җн„° вҖҳл¶„ мӢңк°„ мқј мӣ” л…„вҖҷмқ„ мқҳлҜён•ңлӢӨ.

л¬ём ңк°Җ лҗҳм—ҲлҚҳ cron test кіјм •

л¬ём ңлҠ” м—¬кё°м—җм„ң л°ңмғқн–ҲлӢӨ.
н…ҢмҠӨнҠё мҪ”л“ңк°Җ л§Өмҡ° ліөмһЎн–ҲлҚҳ кҙҖкі„лЎң лҢҖлһөм Ғмқё лјҲлҢҖл§Ң м Ғмһҗл©ҙ, bash script м•Ҳм—җм„ң flockмңјлЎң лҳҗ лӢӨлҘё лӘ…л №м–ҙл“Өмқ„ мӢӨн–үн•ҳлҠ” кө¬мЎ°мҳҖлӢӨ.

лӢӨмқҢкіј к°ҷмқҖ м—җлҹ¬к°Җ л°ңмғқн–ҲлӢӨ.

/bin/bashлЎң мӢӨн–үн•ҳлқјкі  shebang(#!/bin/bash)мқ„ л„Јм—ҲмқҢм—җлҸ„ /bin/shлЎң мӢӨн–үлҗҳм—ҲлӢӨлҠ” кІғм—җ мқҳл¬ёмқ„ н’ҲкІҢ лҗҳм—ҲлӢӨ.

cronмқҖ default shellлЎң мһ‘м—…мқ„ мӢӨн–үмӢңнӮЁлӢӨ

мҲҳм°ЁлЎҖ н…ҢмҠӨнҠён•ң кІ°лЎ л¶Җн„° л§җн•ҳмһҗл©ҙ, cronмқҖ default shell мҰү, $SHELL -c вҖҳл“ұлЎқлҗң мһ‘м—…вҖҷмқ„ мӢӨн–үмӢңнӮЁлӢӨ.
мқҙ л•Ң $SHELLмқҖ кё°ліём ҒмңјлЎң /bin/shмқҙлӢӨ.
мқҙлҠ” л№„лӢЁ cronлҝҗ м•„лӢҲлқј flockлҸ„ к·ёлҹ° кІғмңјлЎң ліҙмқҙлҠ”лҚ°, м•„лһҳмқҳ н…ҢмҠӨнҠё кІ°кіјм—җм„ң м•Ң мҲҳ мһҲлӢӨ.

cronмқҙ мң„ мҠӨнҒ¬лҰҪнҠёлҘј мӢӨн–үн•ҳкІҢ н•ң кІ°кіјлҠ” лӢӨмқҢкіј к°ҷлӢӨ.(л¶Ҳн•„мҡ”н•ң лқјмқёмқҖ мӮӯм ңн–ҲлӢӨ.)

мҡ°м„  cronмқҖ /bin/sh -c вҖҳлӘ…л №вҖҷмңјлЎң мӢӨн–үмӢңнӮЁлӢӨ.
/bin/shмқҖ -cмқҳ мқёмһҗлЎң мҳЁ вҖҳлӘ…л №вҖҷмқ„ мғҲлЎңмҡҙ н”„лЎңм„ёмҠӨлҘј лқ„мӣҢм„ң мӢӨн–үмӢңнӮӨл©°, мқҙ л•Ң shebangмқ„ мқёмӢқн•ҳм—¬ м•Ңл§һлҠ” shellлЎң мӢӨн–үмӢңнӮЁлӢӨ.
flock м—ӯмӢң /bin/sh -c вҖҳлӘ…л №вҖҷмңјлЎң мӢӨн–үмӢңнӮӨлҠ” кІғмқ„ м•Ң мҲҳ мһҲлӢӨ.

мқҙм   crontabм—җ SHELLмқ„ /bin/bashлЎң л“ұлЎқн•ң л’Ө, лҳ‘к°ҷмқҖ мһ‘м—…мқ„ мҲҳн–үн•ҙліҙм•ҳлӢӨ.

м•„лһҳлҠ” мӢӨн–ү кІ°кіјлӢӨ.(м—ӯмӢң л¶Ҳн•„мҡ”н•ң лқјмқёмқҖ мӮӯм ңн–ҲлӢӨ.)

cronлҸ„ flockлҸ„ -c лӢӨмқҢм—җ мҳӨлҠ” лӘ…л №мқ„ мқҙлІҲм—җлҠ” /bin/bashлЎң мӢӨн–үмӢңмј°лӢӨ!

ubuntuмқҳ default shell

лҳ‘к°ҷмқҖ мҠӨнҒ¬лҰҪнҠёлҘј н•„мһҗк°Җ мӢӨн–үн–Ҳмқ„ л•ҢмҷҖ cronмқҙ мӢӨн–үн–Ҳмқ„ л•Ң кІ°кіјк°Җ лӢ¬лһҗлҚҳ кІғмқҖ, кІ°көӯ мӢӨн–үн•ҳлҠ” shellмқҙ лӢ¬лһҗкё° л•Ңл¬ёмқҙм—ҲлӢӨ.

ubuntuмқҳ кІҪмҡ° default shellмқҙ /bin/shмқҙл©°, лі„лӢӨлҘё мҲҳм •мқ„ н•ҳм§Җ м•Ҡм•ҳлӢӨл©ҙ dash(bourne shell)лҘј к°ҖлҰ¬нӮЁлӢӨ.

к·ёл Үкё° л•Ңл¬ём—җ н…ҢмҠӨнҠё кіјм •м—җм„ң cronкіј flockмқҖ ubuntuмқҳ default shellмқё /bin/shлЎң лӘ…л №м–ҙлҘј мӢӨн–үн•ң кІғмқҙлӢӨ.
мқҙ л•Ң /bin/sh -c вҖҳлӘ…л №м–ҙвҖҷлҠ” вҖҳлӘ…л №м–ҙвҖҷлҘј мӢӨн–үн•ҳкё° мң„н•ҙ мғҲлЎңмҡҙ н”„лЎңм„ёмҠӨлҘј лқ„мҡ°л©°, loaderк°Җ shebangмқ„ нҷ•мқён•ңлӢӨ.
(н•„мһҗлҠ” көімқҙ flock -cмқҳ мқёмһҗлЎң мҠӨнҒ¬лҰҪнҠё лӮҙмҡ©мқ„ мӨҢмңјлЎңмҚЁ мӮҪм§Ҳмқ„ н–ҲлҚҳ кІғмқҙм—ҲлӢӨ.
мІҳмқҢл¶Җн„° flock -c вҖҳ#!/bin/bashк°Җ м¶”к°Җлҗң мҠӨнҒ¬лҰҪнҠё нҢҢмқјвҖҷмқ„ мӢӨн–үн•ҳкІҢ н–Ҳмңјл©ҙ кІӘм§Җ м•Ҡм•ҳмқ„ л¬ём ңмҳҖлӢӨвҖҰ!)

л°ҳл©ҙ userмқҳ default login shellмқҙ bashмқҙкё° л•Ңл¬ём—җ, н•„мһҗлҠ” shebangмқ„ м¶”к°Җн•ҳм§Җ м•Ҡкі лҸ„ bashлЎң мҠӨнҒ¬лҰҪнҠёлҘј мӢӨн–үн•  мҲҳ мһҲм—ҲлҚҳ кІғмқҙлӢӨ.

кІ°лЎ 

лҸҢкі  лҸҢм•„мҷ”мңјлӮҳ, мқҙлІҲ н…ҢмҠӨнҠёлЎң н•Ём¶•лҗҳлҠ” лӮҙмҡ©мқҖ лӢӨмқҢ л„Ө к°Җм§ҖмқҙлӢӨ.

  1. ubuntuмқҳ default shellмқҖ /bin/shмқҙл©°, мқҙлҠ” dashлҘј к°ҖлҰ¬нӮЁлӢӨ.
  2. мҠӨнҒ¬лҰҪнҠёлҘј мӢӨн–үн•ҳл©ҙ loaderк°Җ л§Ё мІ« мӨ„мқҳ shebangмқ„ ліҙкі  м•Ңл§һмқҖ shellмқ„ мӢӨн–үмӢңнӮЁлӢӨ.
  3. userмқҳ default login shellмқҖ /bin/bashмқҙлӢӨ.
  4. cronмңјлЎң bash scriptлҘј мӢӨн–үн•ҳкі  мӢ¶лӢӨл©ҙ, мҠӨнҒ¬лҰҪнҠём—җ shebangмқ„ м¶”к°Җн•ҳкұ°лӮҳ crontabм—җ SHELL=/bin/bashлҘј м¶”к°Җн•ҙм•ј н•ңлӢӨ.(мқҙлҠ” cronмқ„ л– лӮҳм„ң м¶”к°Җн•ҙмЈјлҠ” кІғмқҙ м •мӢ кұҙк°•м—җ мўӢлӢӨ.)

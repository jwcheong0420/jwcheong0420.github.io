AWS S3м—җм„ң bucketмқҳ corsлҘј нҷңм„ұнҷ”н•ҳлҠ” л°©лІ•(awscli мӮ¬мҡ©)

Posted on 2018-06-26 Edited on 2020-02-15 In cloud

нҡҢмӮ¬м—җм„ң ceph object storageлҘј мӮ¬мҡ© мӨ‘мқёлҚ°, object м—…лЎңл“ңлҘј мң„н•ҙ frontendлӢЁм—җм„ң http OPTIONSк°Җ м„ұкіөн•ҳлҠ”м§ҖлҘј мІҙнҒ¬н•ңлӢӨкі  н•ңлӢӨ.
мқҙ optionsк°Җ м„ұкіөн•ҳл Өл©ҙ bucketм—җ лҢҖн•ҙ corsлҘј нҷңм„ұнҷ”н•ҙмЈјм–ҙм•ј н•ҳкі , мқҙлҘј aws cliлЎң н•ҳл©ҙ к°„лӢЁн•ҳкІҢ м„ұкіөн–Ҳм§Җл§Ң java мҪ”л“ңлЎң м„ұкіөн•ҳкё°к№Ңм§Җ л§Өмҡ° нһҳл“Өм—ҲлӢӨ.
(м•„м§ҒлҸ„ мқҳл¬ёмқё м җмқҖ, көімқҙ corsлҘј нҷңм„ұнҷ”н•ҳм§Җ м•Ҡм•„лҸ„ object м—…лЎңл“ңлҠ” лҗңлӢӨлҠ” м җмқҙлӢӨ. optionsл§Ң мӢӨнҢЁн•  лҝҗвҖҰ)
aws cliлЎң мӢңлҸ„н–Ҳмқ„ л•Ң вҖ“debugлЎң лӮҳмҳЁ кІ°кіјк°Җ мӢӨм ң aws к°Җмқҙл“ңмҷҖ м•Ҫк°„ лӢӨлҘҙкё°лҸ„ н–Ҳкі , aws signature лІ„м „мқҙ мҳ¬лқјк°Җл©ҙм„ң authorizationмқ„ мғқм„ұн•ҳлҠ” кіјм •мқҙ л§Өмҡ°л§Өмҡ° ліөмһЎн•ҙмЎҢкё° л•Ңл¬ёмқҙм—ҲлӢӨ.
нҳ№мӢңлӮҳ мқҙкІғ л•Ңл¬ём—җ кіЁм№ҳ м•„н”Ң лҜёлһҳмқҳ л¶„мқ„ мң„н•ҙ лӮҙк°Җ л©°м№  лҸҷм•Ҳ мЎ°мӮ¬н•ҳкі  н…ҢмҠӨнҠён•ң лӮҙмҡ©мқ„ кё°лЎқн•ҙл‘җкі мһҗ н•ңлӢӨ.

мҡ°м„  мқҙлІҲ нҸ¬мҠӨнҢ…м—җм„ңлҠ” aws s3, object storage, bucket, cors л“ұл“ұмқҳ к°ңл…җмқ„ к°„лӢЁк°„лӢЁн•ҳкІҢ м„ӨлӘ…н•ҳкі ,
put bucket corsлҘј awscliлЎң н•ҳлҠ” л°©лІ•м—җ лҢҖн•ҙ м„ӨлӘ…н•ҳлҸ„лЎқ н•ҳкІ лӢӨ.
мӣҗлһҳлҠ” AWSv4 Authorization мғқм„ұ л°©лІ•к№Ңм§Җ лӢӨлЈЁл Ө н–ҲлҠ”лҚ°, м“°лӢӨліҙлӢҲ л„Ҳл¬ҙ кёём–ҙм ём„ң мқҙкұҙ лӢӨмқҢм—җ нҸ¬мҠӨнҢ…н•ҙм•јкІ лӢӨ.

AWS S3лһҖ

AWS(Amazon Web Service)м—җм„ң м ңкіөн•ҳлҠ” Object StorageлЎң, Simple Storage ServiceлҘј мӨ„м—¬ S3 м„ңл№„мҠӨлқјкі  н•ңлӢӨ.
Object StorageлҠ” rest apiлҘј мӮ¬мҡ©н•ҳм—¬ objectлҘј key-valueлЎң кҙҖлҰ¬н•ҳлҠ” м ҖмһҘмҶҢлқјкі  мғқк°Ғн•ҳл©ҙ лҗңлӢӨ.
Amazonм—җм„ңлҠ” S3м—җ лҢҖн•ҙ rest apiмҷҖ к°Ғмў… sdk, command lineмңјлЎң мЎ°мһ‘н•ҳлҠ” aws cliлҸ„ м ңкіөн•ҳкі  мһҲлӢӨ.
sdkлҠ” лӮҙл¶Җм ҒмңјлЎң rest apiлҘј мӮ¬мҡ©н•ҳлҜҖлЎң лӢӨлҘј л°”к°Җ м—Ҷкі , cliлҸ„ вҖ“debugлЎң нҷ•мқён•ҙліҙл©ҙ лі„л°ҳ лӢӨлҘҙм§Җ м•ҠмқҢмқ„ м•Ң мҲҳ мһҲлӢӨ.
к·ёлҹ¬лӢҲк№Ң кІ°лЎ мқҖ нҺёмқҳм„ұмқ„ мң„н•ҙ м—¬лҹ¬ л°©лІ•мқҙ м ңкіөлҗҳм§Җл§Ң лӘЁл“  мЎ°мһ‘мқҖ rest apiлЎң к°ҖлҠҘн•ҳлӢӨлҠ” кІғмқҙлӢӨ.

BucketмқҙлһҖ

AWS S3мқҳ кІҪмҡ° userлӢ№ object storageк°Җ л§Ңл“Өм–ҙм§Җкі , к°Ғ object storageлӢ№ access keyмҷҖ secret keyк°Җ л°ңкёүлҗңлӢӨ.
мқҙ л•Ң л‘җ keyлҠ” к°Ғк°Ғ idмҷҖ pwкІ©мңјлЎң мӮ¬мҡ©лҗҳл©°, aws authorization stringмқ„ мғқм„ұн•  л•Ң н•„мҡ”н•ҳлӢӨ.
Object storageлҘј мғқм„ұн–ҲлӢӨкі  н•ҙм„ң л°”лЎң objectлҘј м—…лЎңл“ңн•  мҲҳ мһҲлҠ” кұҙ м•„лӢҲлӢӨ.
л¬јлЎ  object storageлҠ” мӣҗлһҳ objectлҘј flatн•ҳкІҢ кҙҖлҰ¬н•ҳм§Җл§Ң, awsм—җлҠ” bucketмқҙлқјлҠ” layerк°Җ мһҲлӢӨ.
мҰү, object storageм—җ м—¬лҹ¬ к°ңмқҳ bucketмқ„ л§Ңл“Өкі , мқҙ bucketм—җ objectм—җ м—…лЎңл“ңн•ҳлҠ” кө¬мЎ°мқҙлӢӨ.

CORSлһҖ

CORS(Cross-Origin Resource Sharing)мқҖ лӢӨлҘё лҸ„л©”мқём—җ мһҲлҠ” м„ңлІ„мқҳ urlмқ„ нҳём¶ңн•  мҲҳ мһҲлҸ„лЎқ м ‘к·јмқ„ н—Ҳмҡ©н•ҳлҠ” кІғмқҙлӢӨ.
к·ёлҹ¬лӢҲк№Ң corsлҘј м„Өм •н•ҳм§Җ м•Ҡм•ҳлӢӨл©ҙ мӣҗлһҳлҠ” к°ҷмқҖ лҸ„л©”мқём—җм„ңл§Ң м ‘к·јмқҙ к°ҖлҠҘн•ҳкі , лӢӨлҘё лҸ„л©”мқём—җ м ‘к·јн•ҳл Өл©ҙ ліҙм•Ҳ мқҙмҠҲк°Җ л°ңмғқн•ҳлҠ” кІғмқҙлӢӨ.
awsмқҳ кІҪмҡ° bucketм—җ лҢҖн•ҙ cors нҷңм„ұнҷ”лҘј н•ҙмЈјм–ҙм•ј лӢӨлҘё лҸ„л©”мқём—җм„ңлҸ„ м ‘к·јн•  мҲҳ мһҲлӢӨ.

Put bucket cors л°©лІ•

AWS cliлҘј мӮ¬мҡ©н•ҳлҠ” л°©лІ•

  1. aws cliлҘј м„Өм№ҳн•ңлӢӨ.

  2. access keyмҷҖ secret keyлҘј м„Өм •н•ңлӢӨ.

    regionмқҖ м„Өм • м•Ҳн•ҙлҸ„ лҗңлӢӨ. aws configure мғҒм—җм„ңлҠ” кіөлһҖмңјлЎң лҗҳм–ҙмһҲм§Җл§Ң, мӢӨм ң authorization мғқм„ұ мӢң defaultлҠ” us-east-1лЎң мӮ¬мҡ©н•ңлӢӨ.
    output formatмқҖ мһҗмң .

  3. bucketмқ„ мғқм„ұн•ңлӢӨ.

    endpoint-urlмқҖ cephмқҳ object storage gateway ipлҘј м Ғмңјл©ҙ лҗңлӢӨ.
    мқҙкұё мғқлһөн•  кІҪмҡ° s3.amazonaws.comмңјлЎң rest apiлҘј м „мҶЎн•ңлӢӨ.
    м•„ л¬јлЎ  regionмқҙ м„Өм •лҗҳм–ҙмһҲмқ„ кІҪмҡ° urlмқҙ лӢ¬лқјм§Ҳ мҲҳ мһҲлӢӨ.

  4. cors rule м§Җм •

    PUT, GET, DELETE methodм—җ лҢҖн•ҙ нҷңм„ұнҷ”н•ҙлҶ“лҸ„лЎқ м§Җм •н•ҳмҳҖлӢӨ.
    objectлҠ” multipart uploadк°Җ м•„лӢҢ мқҙмғҒ postлҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡкё° л•Ңл¬ёмқҙлӢӨ.

  5. testbucketмқҳ corsлҘј нҷңм„ұнҷ”мӢңнӮЁлӢӨ.

    мқҙм ң testbucketм—җ objectлҘј м—…лЎңл“ңн•ҳкі , м—…лҚ°мқҙнҠён•ҳкі , лӢӨмҡҙлЎңл“ңн•ҳкі , мӮӯм ңн•  мҲҳ мһҲлӢӨ.

AWSv4 Authorization мғқм„ұн•ҳлҠ” л°©лІ•кіј мқҙкұёлЎң http requestлҘј лӮ л Өм„ң bucket corsлҘј нҷңм„ұнҷ”н•ҳлҠ” л°©лІ•мқҖ лӢӨмқҢ нҸ¬мҠӨнҢ…м—җвҖҰ!

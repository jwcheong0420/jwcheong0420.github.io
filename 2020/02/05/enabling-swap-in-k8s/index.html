Kubernetesм—җм„ң swap мӮ¬мҡ©н•ҳлҠ” л°©лІ•

Posted on 2020-02-05 In cloud , k8s

kubernetesлҠ” swapмқ„ мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” кІғмқ„ к¶ҢмһҘн•ңлӢӨ

  • мқҙлҹ¬н•ң мқҙмң л“Ө л•Ңл¬ёмқёлҚ°вҖҰ
    • мҡ”м•Ҫн•ҳмһҗл©ҙ, QoS к°ңл…җкіј мғҒм¶©лҗ  лҝҗлҚ”лҹ¬ swapмқ„ м§Җмӣҗн•ҳл Өл©ҙ кі л Өн•ҙм•јн•  кІҢ л§ҺмңјлҜҖлЎң лӢ№мһҘмқҖ мғқк°Ғн•ҳм§Җ м•ҠкІ лӢӨлҠ” мһ…мһҘ
  • swapмқ„ л¬ҙмӢңн•ҳлҠ” мҳөм…ҳмқҖ deprecated мҳҲм •
    • 1.17.1 лІ„м „м—җм„ң лҸҷмһ‘н•ҳлҠ” кІғмқ„ нҷ•мқён–ҲмңјлӮҳ, м–ём ң м§Җмӣҗмқ„ мӨ‘лӢЁн•  м§Җ лӘЁлҰ„

к·ёлҹјм—җлҸ„ л¶Ҳкө¬н•ҳкі  swapмқ„ мҚЁліҙкІ лӢӨл©ҙвҖҰ

  • н…ҢмҠӨнҠён•ң kubernetes лІ„м „

kubernetes кө¬м„ұн•  л•Ң swapмқ„ л¬ҙмӢңн•ҙліҙмһҗ

  • swap нҢҢнӢ°м…ҳ л§Ңл“Өкі  мјңкё°
  • kubelet config ліҖкІҪ нӣ„ м Ғмҡ©
    • kubeadm init м „м—җлҠ” /var/lib/kubelet/config.yamlмқҙ м—ҶлӢӨлҠ” л©”м„ём§Җк°Җ м¶ңл Ҙлҗҳл©ҙм„ң kubelet мһ¬кё°лҸҷмқҙ лҗҳм§Җ м•ҠлҠ”лӢӨ.
      kubeadm initн•  л•Ң м–ҙм°Ён”ј kubelet кё°лҸҷн•ҳлҠ” лӢЁкі„к°Җ мһҲмңјлҜҖлЎң мқјлӢЁ config м Ғмҡ©к№Ңм§Җл§Ң н•ңлӢӨ.
  • kubeadm init мӢң вҖ“ignore-preflight-errors мҳөм…ҳ м¶”к°Җ
    • мӮ¬мҡ©н•  CNIм—җ л”°лқј м¶”к°Җн•ҙм•јн•ҳлҠ” мҳөм…ҳмқҙ лӢӨлҘҙлӢӨ. flannelмқ„ мӮ¬мҡ©н•ҳлҠ” мҳҲмӢңмқҙлӢӨ.
    • kube configлҘј ліөмӮ¬(kubeadm init кІ°кіјлҘј мһҳ ліөмӮ¬н•ҙл‘ҳ кІғ)
  • kubelet мғҒнғң нҷ•мқё
  • worker nodeлҸ„ swapмқ„ мјңкі  kubelet configлҘј л°”кҫёкё°
  • node join мӢң вҖ“ignore-preflight-errors=Swap м¶”к°Җ

мқҙлҜё кө¬м„ұн•ң kubernetesм—җм„ң swapмқ„ мјңліҙмһҗ

  • worker2мқҳ kubelet мғҒнғң нҷ•мқё
  • swapon /swapfile нӣ„ kubelet мһ¬кё°лҸҷ
  • kubelet configure нҢҢмқј мҲҳм •

kubeletм—җм„ң swap checkн•ҳлҠ” мҪ”л“ң

  • вҖ“fail-swap-onмқҖ defaultлЎң true
    • мҳөм…ҳ мӮ¬мҡ© мӢң кІҪкі  л¬ёкө¬лЎң deprecatedлҗҳм—ҲлӢӨкі  лӮҳмҳӨм§Җл§Ң м•„м§Ғ лҸҷмһ‘н•Ё
  • вҖ“fail-swap-onмқҙ trueмқё кІҪмҡ°, /proc/swapsлҘј мқҪм–ҙм„ң swap м—¬л¶Җ нҷ•мқё (мҶҢмҠӨмҪ”л“ң)
      • swapмқ„ мӮ¬мҡ©н•  кІҪмҡ°, /proc/swapsм—җм„ң л‘җ лІҲм§ё мӨ„мқҙ мЎҙмһ¬н•Ё
      • swapмқ„ мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” кІҪмҡ°, л‘җ лІҲм§ё мӨ„мқҖ мЎҙмһ¬н•ҳм§Җ м•ҠмқҢ
